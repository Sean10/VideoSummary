会议纪要

会议时间： 2023年11月（具体日期未提及）

参会人员： - Ceph 研发人员、字幕翻译及总结人员等

会议内容：

一、Ceph 相关议题

  1. 并行压缩问题：
    • 用户反馈 RocksDB 损坏可能与并行压缩参数更改有关。研发人员已创建不合并分支调查问题。
  2. 深度清理问题：
    • 用户报告即使关闭深度清理，仍有大量深度清理进行。建议增加深度清理的睡眠时间并逐步增加。
  3. 范围删除问题：
    • 用户反馈范围删除导致性能下降和内存碎片。建议在删除大量对象时禁用范围删除并进一步调查。
  4. 删除范围与RocksDB损坏问题：
    • 研发人员表示删除范围可能与RocksDB损坏问题有关。建议在Nautilus版本中禁用删除范围并回退到RocksDB旧版本。
  5. PG数量问题：
    • 讨论了增加PG数量可能带来的问题，建议在减少PG数量时增加PG日志长度。

二、后续行动计划

  • 研发人员将继续调查并行压缩、深度清理、范围删除和RocksDB损坏问题。
  • 用户将根据建议调整深度清理和范围删除的设置。
  • 研发人员将考虑更改PG数量和PG日志长度的默认值。

三、其他事项

  • 讨论了自动缩放器、数据移动和AWS环境等议题。

四、总结

本次会议讨论了Ceph中的一些重要问题，并制定了后续行动计划。研发人员将继续努力解决这些问题，并提高Ceph的性能和稳定性。
